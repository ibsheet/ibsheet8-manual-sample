TypeFormula 데모 — 컬럼 Type 동적 변경 + Type별 보조 속성 셋팅

TextData(회사명) 값에 따라 두 컬럼의 Type이 동적으로 결정됩니다.

① ComboData (Text 출발)

  • 삼성 → Text (보조 속성 불필요)
  • Amazon → Enum (셀별 Enum/EnumKeys 셋팅)
  • Apple → Float (Format 셋팅으로 소수점 표시)
  • Google → Date (Format 3종 + 데이터 timestamp 변환)

② Status (Bool 출발)

  • 대부분 회사: Bool 유지 (체크박스 그대로)
  • Microsoft: Text 전환 → 값을 "활성"/"비활성" 라벨로 변환
핵심 패턴: fr.Row[컬럼명 + 속성명] = 값 형식으로 셀 단위 attr 동적 셋팅 + 필요 시 데이터 값 자체도 변환.
